## Run and build

```bash
# install dependencies
$ npm install

# list available scripts
$ npm run

# serve with hot reload at localhost:3000
$ npm run dev

# build for production
$ npm run build

# Run production build
$ npm run start

# Compile API docs in doc directory
$ npm run gen:doc

```
